Projects



myWeather

A slick weather widget for the Android OS. The core features integrates Google's location API and forecast.io as the data source. The widget also utilizes periodic background services to update data for the end user.

alphasqaures [WIP]

A fun reactional game for the Android OS. Gameplay involves a letter being displayed and users tapping on the corresponding square to earn points. Users have to be quick or else they lose points. This game utilizes Parse's APIs to handle user data and leaderboard data.

FrankerZ

Micromanaging pets for the mobile web. This web game allows users to play with their pets and watch it grow over time. The back-end for the game utilized node.js/PostgreSQL. Crafty.js was then used as the core front-end framework for the gameplay.
